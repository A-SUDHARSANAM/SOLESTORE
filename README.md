# SoleStore Frontend Demo

A lightweight React + Vite shoe store demo. The app is fully frontend-based and uses `localStorage` for cart, wishlist, checkout orders, and dummy order tracking.

## Features

- Men, Women, and Kids product categories
- Product search and filters
- Persistent cart with quantity controls and total price
- Persistent wishlist
- Checkout form with order summary
- Payment simulation with Cash on Delivery or demo online payment
- Dummy order tracking: Order Placed, Packed, Shipped, Out for Delivery, Delivered

## Run

```bash
npm install
npm run dev
```

No backend, database, authentication server, or real payment gateway is required.
